=== audiolize.net Widget for WordPress ===

Automatically injects audiolize.net functionality in WordPress articles.

== Usage ==

1. Install the plugin using WordPress administrative panel.
1. Go to Settings -> audiolize.net section to configure the widget's theme that you want to use.
1. Add `<?php audiolize_widget();?>` within the WordPress theme's content.php 
1. Sign in your audiolize.net content provider account and open any article in your WordPress website. The audiolize.net widget should be visible on the page.

**Example:**

In content.php of your default theme and add `<?php audiolize_widget();?>` immediately before `<div class="entry-content">`.
